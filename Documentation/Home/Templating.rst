==========
Templating
==========

The Fluid Templating Engine
===========================

Fluid is TYPO3’s templating engine. Fluid acts as the glue between your static HTML templates and the content you create in TYPO3’s backend. A
typical Fluid template will contain HTML that defines the structure of your page and Fluid tags that call various types of dynamic content.
