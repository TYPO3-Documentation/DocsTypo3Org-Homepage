==========
Templating
==========


.. container:: row m-0 p-0

   .. container:: col-md-6 pl-0 pr-3 py-3 m-0

      .. container:: card px-0 h-100

         .. rst-class:: card-header

            .. rubric:: :ref:`The Fluid Templating Engine <t3coreapi:fluid>`

         .. container:: card-body

            Fluid is TYPO3’s templating engine. Fluid acts as the glue between your static HTML templates and the
            content you create in TYPO3’s backend. A typical Fluid template will contain HTML that defines the
            structure of your page and Fluid tags that call various types of dynamic content.
