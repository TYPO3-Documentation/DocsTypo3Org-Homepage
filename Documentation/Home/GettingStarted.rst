===============
Getting Started
===============


Installation Guide
------------------

The Installation Guide details all of the steps required to install TYPO3 as well as providing information on how to configure
the host operating system, web server and database.

Site Setup
----------

Once TYPO3 is installed, the next step is to configure a site. The Site Setup Guide explains how to add a domain, configure languages
and set up error handling.

Backend User Management
-----------------------

Details how to create additional backend users and manage their privileges.

Installing Extensions
---------------------

Information on how to install and enable extensions in TYPO3 including built-in core extensions and third party extensions.
